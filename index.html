Git Commands Reference

Initialize & Clone

git init : Initialize a local git repo
git clone https://git@github.com/[username]/[repository-name].git : Create a local copy of repo

Add & Commit

git status : Checks for changes in branches 
git add [file-name] : Add a file to the stage
git add . : Add all changed files to stage 

git commit -m "[commit message]" : Save changes and write commit message

git rm -r [file-name] : Remove a file or folder

Branch & Merge

git branch : List branches 
git brancch -d [branch name] : Delete a branch

git checkout -b [branch name] : Create a new branch and switch to it
git checkout [branch name] : Switch to a branch

git merge [branch name] : Merge a branch into the active branch
git merge [source branch] [target branch] : Merge a branch into a target branch

Send & Receive

git push origin [branch name] : Push a branch to your remote repository
git push -u origin [branch name] : pushes a specific branch [branch name] to the remote repository(origin)
git push : Push changes to remote repository (active branch)
git push origin --delete [branch name] : Delete a remote branch

git pull : Update local repository to the newest commit
git pull origin [branch name] : Pull changes from remote repository

git remote add origin https://git@github.com/[username]/[repository-name].git : Add a remote repository

Comparison & View Changes

git log : View changes
git log --summary : View changes (detailed)
git log --oneline : View changes (briefly)

git diff [source branch] [target branch] : Preview changes before merging
